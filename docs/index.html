Brix - animation

纯声明式、可扩展性动画系统

MineCraft
Animation
典型配置：
<div class="box animated" bx-animation="on:click;class:swing;wait:500;class:wobble"></div>
点击触发
animation: trigger by click
鼠标经过触发
animation: trigger by mouseenter
自动执行动画
animation: automatic execute
调用方式：
  require([
    'brix/animation',
    'jquery',
    'underscore'
  ], function(Anim, $, _) {
    var options = {} //配置
    var anim = new Anim(options)

    //自定义扩展命令: 移除样式
    anim.register('removeClass', function(step) {
      var node = step.node
      var className = step.param
      var done = step.done

      node.removeClass(className)
      done() //结束执行done
    })

    anim.boot() //启动

  })
内置的命令列表：
  on: //dom事件触发
  class: //添加class
  wait: //延迟等待
  removeClass: //移除样式
  style: //添加内联样式
